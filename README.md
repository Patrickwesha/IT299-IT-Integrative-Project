# 📸 Be Seen Photography

Welcome to the official repository for **Be Seen Photography** — a collaborative web project that lets customers view, edit, and order their photo sessions online. Built by a three-person team, this site features a full user experience from browsing to checkout.

---

## 👥 Team Members
- **Patrick** – Home, About, Contact Pages (Static + Layout)
- **Jordan** – Gallery, Photo Editing, Touch-Up Request Pages (Interactive Features)
- **Deidra** – Cart, Checkout, Confirmation, Account, Support Pages (User + Checkout Flow)
